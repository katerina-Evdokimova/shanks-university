Detailed Description

template<typename T, typename K, typename series_templ>
class series_acceleration< T, K, series_templ >

Base class series_acceleration.

It is not used on it's own, but it is inherited by shanks_transformation and epsilon_algorithm to implement the corresponding methods. The class implementation provides everything needed for construction of an arbitrary series up to n terms and printing out the partial sum, the partial sum after transformation is used, and the difference between the latter and the former.
